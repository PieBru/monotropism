# Ein autistischer Student / eine autistische Studentin, den/die Sie im Studium begleiten

> **Für Behinderten-Begleitkräfte, Mentor:innen, Mitschrifthilfen und Assistenzkräfte** — einen autistischen **Studierenden** (etwa 18–25 Jahre) unterstützen.

## Die Kernidee {.lead}
Autismus bedeutet über den **Monotropismus**, dass sich die Aufmerksamkeit einer Studierenden/eines Studierenden in wenige tiefe, intensive „Tunnel" einschließt. Das Studium ist ein *gewaltiger* Übergang, und viele autistische Studierende **prallen an eine Wand** — Burnout, Angst oder Krise — im ersten oder zweiten Jahr. Als Begleitkraft oder Mentor:in können Sie die stetige, vorhersagbare Beziehung sein, die sie über Wasser hält: klare Kommunikation, geschützter Fokus und frühe Warnung, wenn die Last ins Burnout kippt.

## Was Sie beobachten könnten — und was wirklich dahintersteckt

| Sie könnten sehen… | Was oft tatsächlich passiert |
|---|---|
| Hört auf zu kommen, zu antworten oder zur Selbstfürsorge | **Autistisches Burnout** durch kumulative Studienlast |
| Will alles per E-Mail; meidet Anrufe/Treffen | Eingekanalige Kommunikation senkt die Last |
| Verpasst Fristen, kann Aufgaben nicht beginnen | **Autistische Trägheit** + exekutive Überlastung, keine Faulheit |
| Wird tagelang still oder zieht sich zurück | Shutdown/Erholung unter Überlastung |
| Erst jetzt, in der Krise, erkannt | Ein häufiger Punkt später Erkennung |

## Versuchen Sie dies {.do}
- **Seien Sie die vorhersagbare Beziehung** — gleiche Zeit, gleiche Person, gleiches Format; bestätigen Sie schriftlich.
- **Kommunizieren Sie schriftlich**, wörtlich, eine Aufgabe nach der anderen; helfen Sie, Aufgaben in Schritte zu zerlegen.
- **Schützen Sie Fokus und Interesse** — strukturieren Sie das Lernen um ihre Tunnel, nicht um ständiges Wechseln.
- **Normalisieren Sie reduzierte Last** — eine Fristverlängerung, Verschiebung oder ein Unterbruch ist ein legitimer Erholungsschritt.
- **Kennen Sie die Wegweiser** — Behindertenservice, Wohlbefinden, Mentoring, autistische Studierenden-Communities — und gehen Sie *mit* ihnen.
- **Fragen Sie wörtlich nach**, wenn sie still wird; Burnout braucht reduzierte Last, keine Motivationsrede.

## Vermeiden Sie dies {.dont}
- Überraschungswechsel, Druck zum „Durchziehen" oder allgemeiner Sozialkompetenz-Rat.
- Schweigen oder eine verpasste Sitzung mit „nicht engagiert" gleichsetzen.
- Für sie entscheiden — unterstützen Sie Autonomie und Wahl.

## Wann weitere Hilfe nötig ist {.support}
> Achten Sie auf **autistisches Burnout** (Erschöpfung, Kompetenzverlust, verminderte Toleranz) und Krisenzeichen — eskalieren Sie an das Wohlbefindens-/Behindertenteam und, wo nötig, den Schutz. Eine vorübergehende Lastreduktion ist ein evidenzangemessener Erholungsschritt, kein Scheitern.

## Wenn der/die Studierende eine Frau ist {.gender}
> Spät-erkannte Frauen masken stark; eine „gesellige Überfliegerin" bricht vielleicht privat zusammen. Nehmen Sie verminderte Toleranz und Rückzug ernst, wie „zusammen" sie auch wirkt.
